scientia-plm by scientia

Scientia's Product Lifecycle Management Solution

Scientia PLM is an open source solution for Product Lifecycle Management. It provides part management, document management and change management. With nex-gen user interface, database and os independent , full source code, Scientia PLM stands out from the current available PLM solution.

Dependencies

java, GWT, Grails, spring framework

Install

1. download grails 2. instal grails as per the instructions 3. download the project or clone the repository 4. Run "grails run-app" in the unzipped or cloned project folder 5. At the end grails will show the URL http://localhost:8080/scientia. OR 1. download the war 2. deploy it to your tomcat installation
